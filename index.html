Smart Writing Pad for Children with SLD

An AI-powered web platform that helps children with Specific Learning Differences build handwriting skills through real-time feedback and gamified progress tracking.

  • Web Speech API integration for real-time spoken feedback in learning sessions
  • Progress dashboard with SVG trend charts, badge reward system, and session persistence
  • Python/OpenCV module for webcam-based air-writing recognition
